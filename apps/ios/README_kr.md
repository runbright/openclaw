# OpenClaw iOS (슈퍼 알파 버전)

**주의: 현재 TestFlight를 통한 배포는 제공되지 않습니다.**

이 iPhone 앱은 슈퍼 알파 버전이며 내부 테스트용입니다. OpenClaw 게이트웨이에 `role: node`로 연결됩니다.

## 배포 상태
- 현재 배포 방식: Xcode를 통한 소스 코드 기반 로컬/수동 배포만 가능합니다.
- App Store 출시는 현재 내부 개발 로드맵에 포함되어 있지 않습니다.

## 슈퍼 알파 버전 고지사항
- 파괴적인 변경 사항(Breaking changes)이 자주 발생할 수 있습니다.
- UI 및 온보딩 흐름이 예고 없이 변경될 수 있으며 마이그레이션을 보장하지 않습니다.
- 현재로서는 **포그라운드(앱이 화면에 떠 있는 상태)**에서의 사용만이 가장 안정적입니다.

## Xcode 수동 배포 절차
1. **준비물**: Xcode 16+, `pnpm`, `xcodegen`, Apple 개발자 계정 서명 설정.
2. **저장소 루트에서 실행**:
   ```bash
   pnpm install
   ./scripts/ios-configure-signing.sh
   cd apps/ios
   xcodegen generate
   open OpenClaw.xcodeproj
   ```
3. **Xcode에서**: Scheme을 `OpenClaw`로 설정하고 연결된 iPhone을 대상으로 실행(Run)합니다.

## 주요 기능 (현재 작동하는 것들)
- 설정 코드 흐름을 통한 페어링 (Telegram에서 `/pair` -> `/pair approve`).
- 자동 탐지 또는 수동 호스트 설정을 통한 게이트웨이 연결.
- 오퍼레이터 게이트웨이 세션을 통한 채팅 및 대화(Talk).
- iPhone 노드 명령어(포그라운드): 카메라 촬영, 캔버스 조작, 화면 녹화, 위치 정보, 연락처, 캘린더, 미리 알림, 사진 등.
- 다른 앱에서 공유하기(Share extension)를 통해 게이트웨이 세션으로 링크 전달.

## 알려진 문제 및 제한 사항
- **백그라운드 제한**: iOS 정책상 백그라운드에서는 소켓 연결이 중단될 수 있으며, 재연결 로직을 계속 개선 중입니다.
- 백그라운드 상태에서는 카메라, 화면 녹화, 대화(Talk) 등의 기능이 차단됩니다.
- 위치 정보 자동화를 위해서는 위치 권한을 '항상 허용'으로 설정해야 합니다.
