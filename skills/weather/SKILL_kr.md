# 날씨 스킬

현재 날씨 상태와 일기 예보를 조회합니다.

## 사용 사례

✅ **다음과 같을 때 이 스킬을 사용하세요:**
- "오늘 날씨 어때?"
- "내일 비 와?"
- "[도시 이름] 온도 알려줘"
- "이번 주 일기 예보 보여줘"
- 여행 계획 시 날씨 확인

❌ **다음과 같을 때 이 스킬을 사용하지 마세요:**
- 과거 날씨 데이터 조회 → 전문 기상 아카이브/API 사용
- 기후 분석 또는 추세 파악
- 항공/해양 기상 정보 → 전문 서비스(METAR 등) 사용

## 명령어 예시

### 현재 날씨
```bash
# 한 줄 요약
curl "wttr.in/Seoul?format=3"

# 상세 현재 상태
curl "wttr.in/Seoul?0"
```

### 일기 예보
```bash
# 3일 예보
curl "wttr.in/Seoul"

# 이번 주 예보
curl "wttr.in/Seoul?format=v2"

# 특정 날짜 (0=오늘, 1=내일, 2=모레)
curl "wttr.in/Seoul?1"
```

### 포맷 코드 안내
- `%c`: 날씨 상태 이모지
- `%t`: 온도
- `%f`: 체감 온도
- `%w`: 풍속
- `%h`: 습도
- `%p`: 강수량
- `%l`: 위치
