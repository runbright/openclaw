# TOOLS.md - 로컬 메모

스킬이 도구의 _작동 방식_을 정의합니다. 이 파일은 _당신의_ 구체적인 사항 -- 당신의 환경에 고유한 것들을 위한 것입니다.

## 여기에 넣을 것

다음과 같은 것들:

- 카메라 이름 및 위치
- SSH 호스트 및 별칭
- TTS 선호 목소리
- 스피커/방 이름
- 기기 별명
- 환경별 모든 것

## 예시

```markdown
### 카메라

- living-room → 메인 공간, 180° 광각
- front-door → 현관, 모션 트리거

### SSH

- home-server → 192.168.1.100, user: admin

### TTS

- 선호 목소리: "Nova" (따뜻하고, 약간 영국식)
- 기본 스피커: Kitchen HomePod
```

## 왜 분리하나요?

스킬은 공유됩니다. 당신의 환경은 당신만의 것입니다. 분리하면 메모를 잃지 않고 스킬을 업데이트할 수 있고, 인프라를 유출하지 않고 스킬을 공유할 수 있습니다.
