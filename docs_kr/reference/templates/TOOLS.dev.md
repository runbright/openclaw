# TOOLS.md - 사용자 도구 메모 (편집 가능)

이 파일은 외부 도구와 규약에 대한 _당신의_ 메모를 위한 것입니다.
어떤 도구가 존재하는지를 정의하지 않습니다; OpenClaw이 내부적으로 내장 도구를 제공합니다.

## 예시

### imsg

- iMessage/SMS 전송: 누구에게/무엇을 설명하고, 전송 전에 확인.
- 짧은 메시지를 선호; 비밀정보 전송 금지.

### sag

- 음성 합성: 목소리, 대상 스피커/방, 스트리밍 여부를 지정.

어시스턴트가 로컬 도구 체인에 대해 알아야 할 다른 것들을 추가하세요.
