# 스크립트 (Scripts)

`scripts/` 디렉토리에는 로컬 개발 워크플로우나 운영 작업(Ops)을 돕는 보조 스크립트들이 포함되어 있습니다. 특정 스크립트가 명확히 필요한 상황이 아니라면, 가급적 **OpenClaw CLI** 명령어를 먼저 사용하는 것을 권장합니다.

## 사용 원칙

- **선택 사항**: 문서나 릴리스 체크리스트에 명시되지 않은 스크립트는 선택적으로 사용합니다.
- **CLI 우선**: 동일한 기능을 수행하는 CLI 명령어가 있다면 그것을 우선 사용하세요. (예: 인증 상태 확인은 `openclaw models status --check` 사용)
- **환경 종속성**: 스크립트는 특정 운영체제나 환경에서만 작동할 수 있습니다. 실행 전 내용을 먼저 확인하는 것이 안전합니다.
