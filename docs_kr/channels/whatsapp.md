# 왓츠앱 (WhatsApp Web Channel)

왓츠앱 채널은 왓츠앱 웹(WhatsApp Web) 프로토콜을 사용하여 연동됩니다. 컴퓨터에서 왓츠앱 웹에 로그인하는 것과 같은 방식으로 QR 코드를 스캔하여 에이전트를 연결할 수 있습니다.

## 빠른 설정 방법

1.  **권한 설정**: `openclaw.json` 파일에서 왓츠앱 정책을 설정합니다. (처음에는 `pairing` 모드를 권장합니다.)
2.  **로그인 (QR 코드)**: 터미널에서 다음 명령어를 입력하면 화면에 QR 코드가 나타납니다. 휴대폰의 왓츠앱 앱에서 '연결된 기기'를 통해 스캔하세요.
    ```bash
    openclaw channels login --channel whatsapp
    ```
3.  **게이트웨이 실행**: `openclaw gateway`를 실행하여 연결을 유지합니다.
4.  **페어링 승인**: 봇에게 메시지를 보낸 뒤, 서버 터미널에서 승인 코드를 입력합니다.
    ```bash
    openclaw pairing list whatsapp
    openclaw pairing approve whatsapp <코드>
    ```

## 운영 방식 추천

- **전용 번호 사용 (강력 추천)**: 개인 번호보다는 에이전트 전용의 별도 번호를 사용하는 것이 보안과 관리 측면에서 가장 깔끔합니다.
- **개인 번호 사용**: 본인의 개인 번호를 그대로 연결하여 자신에게 메시지를 보내는 방식으로 사용할 수도 있습니다. 이 경우 설정에서 `selfChatMode: true`를 확인하세요.

## 주요 기능 지원

- **텍스트 및 미디어**: 일반 메시지뿐만 아니라 이미지, 문서 전송도 지원합니다.
- **그룹 대화**: 봇을 그룹에 초대할 수 있으며, 그룹별로 독립된 세션을 가집니다.
- **이미 읽음 처리**: 에이전트가 메시지를 읽으면 휴대폰에서도 읽음 표시가 나타납니다.
