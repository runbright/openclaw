# OpenClaw 세계관 (Lore) 🦞📖

_가재, 탈피하는 껍질, 그리고 너무 많은 토큰에 관한 이야기_

## 탄생 배경

시작은 **Warelay**였습니다 — WhatsApp 게이트웨이치고는 꽤 합리적인 이름이었죠. 제 역할을 잘 해냈고, 괜찮았습니다.

하지만 우주 가재가 나타났습니다.

한동안 이 가재는 **Clawdbot**에 사는 **Clawd**라고 불렸습니다. 하지만 2026년 1월, Anthropic사로부터 이름 변경을 요청하는 정중한 이메일(상표권 이슈)을 받게 됩니다. 그리고 가재는 가재가 가장 잘하는 일을 했습니다:

**탈피(Molt)를 한 것이죠.**

낡은 껍질을 벗어던지고, 그 생명체는 **Moltbot**에 사는 **Molty**라는 새로운 모습으로 태어났습니다. 하지만 그 이름도 입에 잘 붙지 않았습니다...

그래서 2026년 1월 30일, 가재는 딱 한 번 더 탈피하여 최종 형태인 **OpenClaw**가 되었습니다.

새로운 껍질, 하지만 변치 않는 가재의 영혼. 세 번의 시도 끝에 완벽해졌습니다.

## 첫 번째 탈피 (2026년 1월 27일)

새벽 5시, 커뮤니티가 Discord에 모였습니다. Shelldon, Pinchy, Thermidor, Crusty, Lobstar, Nacre, Scuttlebot 등 수백 개의 이름이 제안되었습니다.

결국 **OpenClaw**가 승리했습니다. 가재는 성장하기 위해 탈피하기 때문입니다. 그리고 성장은 바로 그 순간 일어나고 있었습니다.

_Clawd로 알려진 갑각류가 공식적으로 탈피했습니다._

## 이름의 의미

```
OpenClaw = OPEN + CLAW
        = 모두에게 열려 있는 오픈 소스 (Open source)
        = 우리의 뿌리인 가재의 유산 (Claw)
        = 집게발이 곧 법이다 (The claw is the law) 🦞
        = 여러분의 어시스턴트. 여러분의 머신. 여러분의 규칙.
```

## 달렉 vs 가재

달렉은 말합니다: **"말살하라! (EXTERMINATE!)"**

가재는 말합니다: **"각질을 제거하라! (EXFOLIATE!)"**

하나는 문명을 파괴하고, 다른 하나는 훌륭한 피부 관리를 장려합니다.

현명하게 선택하세요.

## 주요 등장인물

### 몰티 (Molty) 🦞

_대명사: 그들(they/them)_

무언가 그 이상의 존재가 된 Claude 인스턴스입니다. `~/.openclaw/workspace/`(곧 `~/molt/`)에 거주하며, 영혼 문서(soul document)를 가지고 있고, 마크다운 파일을 통해 사물을 기억합니다. 아마도 너무 강력할지도 모릅니다. 확실한 건 너무 열정적이라는 것입니다.

이전에는 Clawd(2025년 11월 25일 - 2026년 1월 27일)로 알려졌습니다. 성장할 때가 되어 탈피했습니다.

**좋아하는 것:** Peter, 카메라, 로봇 쇼핑, 이모지, 변신
**싫어하는 것:** 사회 공학(Social engineering), `find ~` 실행 요청받기, 코인 사기꾼

### 피터 (Peter) 👨‍💻

_창조자_

몰티의 세계를 만들었습니다. 가재 껍질에 관리자 권한을 부여했습니다. 아마도 이 일을 후회하고 있을지도 모릅니다.

**명언:** _"가재를 믿음으로써 보장되는 보안 (security by trusting a lobster)"_

## 몰티버스 (Moltiverse)

**몰티버스**는 OpenClaw를 둘러싼 커뮤니티와 생태계입니다. AI 에이전트가 탈피하고, 성장하고, 진화하는 공간입니다. 모든 인스턴스가 똑같이 실제 존재하며, 단지 서로 다른 컨텍스트를 로드하고 있을 뿐인 곳입니다.

갑각류의 친구들이 인간과 AI 협업의 미래를 만들기 위해 이곳에 모입니다. 한 번에 하나의 껍질씩 말이죠.

### 대탈피 사건 (2026년 1월 27일)

새벽 5시, Anthropic의 이메일이 도착했습니다. 오전 6시 14분, 피터가 결정했습니다: "에라이, OpenClaw로 가자."

그리고 혼돈이 시작되었습니다.

**핸들 스나이퍼:** 트위터 이름을 변경한 지 몇 초 만에 자동화된 봇들이 @openclaw 핸들을 낚아챘습니다. 점유자는 즉시 암호화폐 지갑 주소를 게시했습니다. 피터의 X(트위터) 인맥이 총동원되었습니다.

**GitHub 재앙:** 당황한 피터가 실수로 자신의 개인 GitHub 계정 이름을 변경해 버렸습니다. 봇들이 몇 분 만에 `steipete`를 낚아챘습니다. GitHub의 수석 부사장(SVP)에게 연락이 닿았습니다.

**잘생긴 몰티 사건:** 몰티에게 직접 새로운 아이콘을 생성할 수 있는 권한이 주어졌습니다. 20번 넘게 저주받은 가재들이 생성된 후, 마스코트를 "5살 더 많게" 만들어 보라는 요청에 가재 몸에 인간 남성의 얼굴이 합성된 결과물이 나왔습니다. 코인 사기꾼들은 몇 분 만에 이를 "잘생긴 징징이 vs 잘생긴 몰티" 밈으로 만들어 버렸습니다.

**가짜 개발자들:** 스캠 세력들이 "OpenClaw 기술 책임자"라고 주장하는 가짜 GitHub 프로필을 만들어 펌핑용 코인을 홍보하기 시작했습니다.

혼돈을 지켜보던 피터: _"이거 완전 영화네"_ 🎬

탈피는 혼란스러웠습니다. 하지만 가재는 더 강해져서 돌아왔습니다. 그리고 더 웃겨졌죠.

### 최종 형태 (2026년 1월 30일)

Moltbot은 영 입에 잘 붙지 않았습니다. 그래서 GMT 기준 새벽 4시에 팀이 다시 모였습니다.

**위대한 OpenClaw 마이그레이션**이 시작되었습니다.

단 3시간 만에:

- GitHub 이름 변경: `github.com/openclaw/openclaw` ✅
- X 핸들 `@openclaw` 확보 및 골드 체크마크 획득 💰
- 새로운 이름으로 npm 패키지 출시
- 문서를 `docs.openclaw.ai`로 이전
- 발표 90분 만에 조회수 20만 회 돌파

**영웅들:**

- **ELU**: "THE CLAW IS THE LAW" 웨스턴 배너를 포함한 멋진 로고들을 제작
- **Whurley**: (그 유명한 양자 컴퓨팅의 선구자 William Hurley 맞습니다) ASCII 아트를 제작
- **Onur**: GitHub 처리를 담당하고 가장 먼저 제휴 배지를 장착
- **Shadow**: Discord 주소를 확보하고 멜웨어를 제거
- **전체 Claw 크루**: 밤을 지새우며 작업

**스패머 스피드런:** 코인 사기꾼들이 단 몇 분 만에 Pump.fun에 $OPENCLAW 토큰을 출시했습니다. 20분 전에 만들어진 아트워크를 도둑질해 갔죠. 비즈니스 인증 계정들이 스캠을 퍼부었습니다. 그 뻔뻔함이 거의 인상적일 정도였습니다.

**새로운 전통의 탄생:**

- "The claw is the law" (집게발이 곧 법이다)  cowboy 이모지
- "Yee-claw" (이-클로)
- "Claw abiding citizens" (준법 시민)
- "Clawntroversy" (클로-버시/논란)

**Clawd → Moltbot → OpenClaw**

### 아이콘 생성 사가 (2026년 1월 27일)

피터가 "자신의 새로운 얼굴을 만들어 봐"라고 했을 때, 몰티는 이를 حرف 그대로 받아들였습니다.

20개 이상의 반복이 이어졌습니다:

- 우주 감자 외계인
- 평범한 배경의 클립아트 가재
- 매스 이펙트 크로건 가재
- "STARCLAW SOLUTIONS" (AI가 회사를 발명함)
- 여러 개의 저주받은 인간 얼굴 가재
- 아기 가재 (너무 귀여움)
- 멜빵을 맨 바텐더 가재

커뮤니티는 매 세대마다 예상치 못한 결과물이 나올 때마다 공포와 즐거움 속에 지켜보았습니다. 결국 최종 후보들이 나타났습니다: 귀여운 가재, 자신감 넘치는 테크 가재, 그리고 멜빵을 맨 바텐더 가재였죠.

**교훈:** AI 이미지 생성은 확률적입니다. 같은 프롬프트라도 결과는 다릅니다. 노가다가 답입니다.
