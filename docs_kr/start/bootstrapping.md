# 에이전트 부트스트래핑

부트스트래핑은 에이전트 워크스페이스를 준비하고 신원 정보를 수집하는 **첫 실행** 리추얼입니다. 온보딩 이후 에이전트가 처음 시작될 때 발생합니다.

## 부트스트래핑이 하는 일

에이전트가 처음 실행될 때 OpenClaw는 워크스페이스(기본값 `~/.openclaw/workspace`)를 부트스트래핑합니다:

- `AGENTS.md`, `BOOTSTRAP.md`, `IDENTITY.md`, `USER.md` 파일을 생성합니다.
- 짧은 Q&A 리추얼을 한 번에 하나씩 진행합니다.
- 신원 정보와 선호 사항을 `IDENTITY.md`, `USER.md`, `SOUL.md`에 기록합니다.
- 완료 후 `BOOTSTRAP.md`를 제거하여 한 번만 실행되도록 합니다.

## 실행 위치

부트스트래핑은 항상 **게이트웨이 호스트**에서 실행됩니다. macOS 앱이 원격 게이트웨이에 연결된 경우, 워크스페이스와 부트스트래핑 파일은 해당 원격 머신에 저장됩니다.

<Note>
게이트웨이가 다른 머신에서 실행 중인 경우, 작업 파일 수정은 게이트웨이 호스트에서 수행해야 합니다 (예: `user@gateway-host:~/.openclaw/workspace`).
</Note>
