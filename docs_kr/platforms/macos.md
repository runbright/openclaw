# OpenClaw macOS 동반 앱

OpenClaw macOS 앱은 메뉴 바에서 상주하며 시스템 권한을 관리하고, 로컬 게이트웨이와 연결되어 맥의 다양한 기능을 에이전트에게 제공하는 역할을 합니다.

## 주요 기능

- **메뉴 바 상태 표시**: 게이트웨이의 가동 상태를 실시간으로 보여주고 알림을 제공합니다.
- **시스템 권한 관리**: 알림, 접근성, 화면 기록, 마이크 등 맥의 보안 권한(TCC) 승인을 관리합니다.
- **노드 기능 지원**: 맥의 카메라, 화면 기록, 캔버스(웹뷰), 시스템 명령어(`system.run`) 실행을 에이전트가 사용할 수 있도록 '노드(Node)' 역할을 수행합니다.
- **로컬/원격 모드**: 내 컴퓨터에서 직접 게이트웨이를 돌릴 수도 있고(로컬), 외부 서버에 있는 게이트웨이에 연결하여 내 맥을 조작하게 할 수도 있습니다(원격).

## 로컬 모드 vs 원격 모드

- **로컬 (기본값)**: 내 컴퓨터에서 게이트웨이 서비스를 함께 실행합니다. 앱이 시작될 때 게이트웨이 프로세스 상태를 점검하고 필요 시 시작합니다.
- **원격 (Remote)**: 게이트웨이는 외부 서버에서 돌아가고, 내 맥은 오직 에이전트의 명령을 듣는 '기기(Node)'로서만 작동합니다. SSH 터널링이나 Tailscale을 통해 원격 게이트웨이와 안전하게 통신합니다.

## 시스템 명령어 실행 승인 (`system.run`)

보안을 위해 에이전트가 내 맥에서 명령어를 실행할 때마다 승인이 필요할 수 있습니다.
- **설정 위치**: 앱 설정 → Exec approvals
- **작동 방식**: 처음 실행하는 명령어는 팝업을 통해 사용자에게 물어보며, '항상 허용'을 선택하면 승인 목록(Allowlist)에 등록되어 다음부터는 묻지 않습니다.

## 관련 명령어 (CLI)

맥 앱과 통신하거나 상태를 확인하는 데 사용됩니다:
```bash
open 'openclaw://agent?message=안녕' # 딥 링크로 메시지 보내기
openclaw nodes status               # 연결된 맥 노드 상태 확인
```
