# 사용량 추적 (Usage Tracking)

OpenClaw는 사용 중인 LLM 서비스 제공자의 API를 통해 실제 사용량과 할당량(Quota) 정보를 직접 가져와 보여줍니다. 단순히 토큰 수를 계산하는 것이 아니라, 서비스 제공자가 보고하는 정확한 데이터를 기반으로 합니다.

## 확인 방법

### 1. 채팅창 명령어
- `/status`: 현재 세션의 토큰 사용량과 예상 비용을 포함한 상태 카드를 보여줍니다.
- `/usage tokens`: 매 답변 하단에 사용된 토큰 수를 표시합니다.
- `/usage full`: 상세한 사용량 통계를 표시합니다.
- `/usage cost`: OpenClaw 세션 로그를 기반으로 합산된 로컬 비용 요약을 보여줍니다.

### 2. CLI 명령어
- `openclaw status --usage`: 모든 모델 제공자별 사용량 정보를 상세히 출력합니다.
- `openclaw channels list`: 등록된 채널 정보와 함께 사용량 스냅샷을 보여줍니다.

### 3. 기타 인터페이스
- **macOS 메뉴 바 앱**: Context 메뉴 하단의 "Usage" 섹션에서 확인할 수 있습니다.

## 지원되는 제공자 및 요건

사용량 정보를 가져오기 위해서는 관리자 권한이나 적절한 인증 정보가 필요합니다:

- **Anthropic (Claude)**: 인증 프로필의 OAuth 토큰이 필요합니다.
- **OpenAI Codex**: OAuth 토큰 또는 계정 ID가 필요합니다.
- **GitHub Copilot**: OAuth 토큰이 필요합니다.
- **MiniMax / z.ai**: 설정된 API 키를 사용합니다.

적절한 인증 정보가 없거나 서비스 제공자에서 사용량 API를 지원하지 않는 경우, 해당 정보는 대화창이나 CLI에서 숨겨집니다.

상세한 기술 사양은 [영문 원본 문서](/concepts/usage-tracking)를 참고하십시오.
