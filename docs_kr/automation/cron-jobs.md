# 크론 작업 (Cron Jobs)

크론(Cron)은 OpenClaw 게이트웨이에 내장된 스케줄러입니다. 작업을 저장해 두었다가 정해진 시간에 에이전트를 깨우고, 필요한 경우 결과를 채팅창에 보내줍니다.

## 주요 특징
- **지속성**: 작업 정보는 `~/.openclaw/cron/`에 저장되어 게이트웨이를 재시작해도 사라지지 않습니다.
- **실행 방식**:
    - **메인 세션**: 기존 대화 흐름에 시스템 이벤트를 추가합니다. (예: "알림: 회의 준비하세요")
    - **격리된 세션**: 독립된 세션에서 특정 작업을 수행하고 결과만 보고합니다.
- **다양한 스케줄**: 한 번만 실행(Reminder), 주기적 반복(Every), 복잡한 크론 표현식(0 7 * * *)을 모두 지원합니다.

## 명령어 예시 (CLI)

### 1회성 알림 추가
```bash
openclaw cron add \
  --name "리마인더" \
  --at "2026-03-01T09:00:00Z" \
  --system-event "서류 제출 확인하기" \
  --wake now \
  --delete-after-run
```

### 매일 아침 반복 작업 추가
```bash
openclaw cron add \
  --name "아침 브리핑" \
  --cron "0 7 * * *" \
  --session isolated \
  --message "오늘의 주요 뉴스 요약해줘." \
  --announce \
  --channel whatsapp
```

## 주요 용어 설명

- **스케줄 종류**:
    - `at`: 특정 시각에 한 번 실행.
    - `every`: 일정 간격(밀리초)마다 실행.
    - `cron`: 표준 크론 표현식을 사용한 정기 실행.
- **세션 타겟**:
    - `main`: 지금 대화 중인 채팅방에 끼어듭니다.
    - `isolated`: 배경에서 조용히 작업하고 결과만 전달합니다.
- **웨이크 모드 (Wake Mode)**:
    - `now`: 시간이 되면 즉시 에이전트를 깨워 실행합니다.
    - `next-heartbeat`: 다음 주기적 확인 시간에 함께 처리합니다.

## 관리 명령어
- `openclaw cron list`: 설정된 모든 작업 보기
- `openclaw cron run <ID>`: 특정 작업을 지금 즉시 실행
- `openclaw cron delete <ID>`: 작업 삭제
